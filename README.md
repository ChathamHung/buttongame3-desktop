# Desktop app for Button Game 3
Button Game 3 Desktop App allow you play the game on your desktop, not just in the browser! It have clear UI and same game content in desktop.
The apps are bulid by [Wails](https://wails.io). But in the futher, I think I will switch to [Tauri](https://tauri.app/).

## Button Game 3
To play the game online, please click [here](https://chathamhung.github.io/buttongame3) or vist on [Github](https://github.com/ChathamHung/buttongame3).

## Notes
The desktop app are the first version, so it includes lot of bugs, please report the bugs on [issues page](https://github.com/ChathamHung/buttongame3-desktop/issues) when you found bugs. Thanks you.
