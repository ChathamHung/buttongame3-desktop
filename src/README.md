## 🌟 Overview

Button Game 3 is an innovative browser-based puzzle game that combines creative challenges, brain teasers, and interactive gameplay. Players must think outside the box to progress through 43 carefully crafted levels, each presenting unique obstacles and puzzles to solve.
In total the game has *50 levels* for you to play!

## 🎯 Game Features

### 🧩 **Diverse Level Types**
- **Hidden Button Challenges** - Find cleverly concealed buttons
- **Interactive Quizzes** - Test your knowledge and reasoning
- **Memory Games** - Challenge your recall abilities
- **Captcha Puzzles** - Prove you're human in creative ways
- **Mini-Games** - Enjoy variety with timers, clicking challenges, and more
- **Code Breaking** - Decrypt messages and solve ciphers
- **Pattern Recognition** - Match colors, shapes, and sequences

### 🎮 **Game Mechanics**
- **43 Unique Levels** - From "Welcome" to "Shape Clicking"
- **Progressive Difficulty** - Levels get more challenging as you advance
- **Save System** - Automatic progress tracking
- **Skip Options** - Get hints or skip difficult levels
- **Level Navigation** - Jump between unlocked levels
- **Achievement System** - Track your accomplishments

### 🎨 **User Experience**
- **Responsive Design** - Works on desktop and mobile devices
- **Intuitive Interface** - Clean, modern UI design
- **Interactive Tips** - Helpful guidance when you're stuck
- **Smooth Animations** - Polished visual feedback
- **No Downloads** - Play instantly in your browser

## 🚀 Getting Started

### 🌐 Play Online (Recommended)
Visit **[Button Game 3](https://chathamhung.github.io/buttongame3/)** and start playing immediately!

### 💻 Play On Desktop
Visit **[Button Game 3 Desktop app](https://chathamhung.github.io/buttongame3-desktop/)** and download it!

### 📂 Local Development
1. **Clone the repository:**
   ```bash
   git clone https://github.com/ChathamHung/buttongame3.git
   cd buttongame3
   ```

2. **Open in browser:**
   - Simply open `index.html` in your preferred web browser
   - No build process or dependencies required!

3. **Start playing:**
   - The game runs entirely client-side
   - Your progress is automatically saved locally
